How to view locally:
1. Open index.html in a browser directly (double-click).
2. Or run a simple local server, e.g.:
   python3 -m http.server 8000
   then open http://localhost:8000

How to deploy to GitHub Pages:
1. Create a GitHub repository (e.g. nikesh.github.io or your-repo-name).
2. Upload all files to the repository root (or push via git).
3. In repository Settings -> Pages, choose the branch (main) and root folder.
4. The site will be available at https://yourusername.github.io/ or https://yourusername.github.io/your-repo-name/

You can replace assets/profile.png with your own photo. Update contact email in index.html as needed.
